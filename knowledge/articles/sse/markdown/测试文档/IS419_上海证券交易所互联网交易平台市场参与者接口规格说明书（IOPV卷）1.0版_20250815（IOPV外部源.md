1. 实时报文格式约定
1.1. 报文通讯模式
获得IOPV 上传资质的指定机构可以通过HTTP 协议的RESTful API 接口接入互联网交易
平台并上传IOPV 数据,通过POST+JSON 方式提交报文, 报文采用UTF-8 编码。

度； NX(Y) 代表该字符串为浮点数， X 代表该字符串的最大长度， Y 代表小数位数，小数部
分最多不超过 Y 位。
